# Enterprise Data Quality Studies (1995–2025): Machine-readable Dataset & Evidence Map (N=106)

This repository provides the machine-readable dataset snapshot used in the Scientific Data (Article) submission:
**“A machine-readable dataset and evidence map for enterprise data quality studies”**.

The archived, citable dataset snapshot is hosted on Zenodo:
**DOI: 10.5281/zenodo.18286291**

---

## What this dataset contains

The Zenodo package (and this repository, if mirrored) includes two Excel files:

- **data.xlsx**  
  Included-study list (**N=106**, 1995–2025) with bibliographic profiling variables and screening outcomes.

- **yjt_data.xlsx**  
  Machine-readable coding table (**one row per study**) including:
  - EMT variables (Evaluation–Method–Tool coding schema)
  - C1–C12 challenge tags with ordinal depth (0/1/2)

> Note: This repository currently provides the dataset files and documentation only.  
> A separate codebook PDF and reproduction scripts are not included at this stage.

---

## File manifest

A structured file manifest is provided in **FILE_MANIFEST.csv** to explain the purpose and scope of each file.

---

## How to reuse

1. Use **data.xlsx** to identify and profile the included studies (N=106).
2. Use **yjt_data.xlsx** for analysis, evidence mapping, and benchmarking based on EMT variables and C1–C12 tags.
3. Cite the Zenodo DOI when reusing the dataset or derived results.
